# family-patch-releases

Public release repository for the Family Patch client applications.

## Purpose
This repository is used only to publish downloadable release binaries and release notes for Family Patch client applications.

## Important
This is a **release-only repository**.
It does **not** contain source code, build scripts, or private server-side implementation.
Design documents, source code, and analysis services are maintained separately in a private repository.

## Platforms
Release assets may be published here for:
- Windows
- macOS

Platform-specific downloads will be attached to each release with clear file names.

Examples:
- `family-patch-windows-x64.zip`
- `family-patch-bootstrap-windows-x64.exe`
- `family-patch-macos-universal.zip`

## How to Download
Go to the **Releases** section of this repository and download the latest approved asset for your platform.

Family members do **not** need a GitHub account to download public release assets.

## Notes
- This repository is intended for software distribution only.
- Do not expect source code to appear here.
- Release notes will describe major changes and installation/update guidance.
